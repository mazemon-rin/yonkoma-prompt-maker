# 選ぶだけイラストプロンプトメーカー

## アプリ概要

リストから選ぶだけで、画像生成AIへそのまま渡せるイラスト用プロンプトを作成できます。

1枚絵と4コマ漫画を切り替えて利用できます。

個人の体験談、学習記録、仕事の気づき、SNS投稿、ブログ挿絵、note記事の挿絵などに使えます。

外部APIは使いません。HTML、CSS、JavaScriptだけでブラウザ上で動作します。

## note購入者特典としての使い方

このアプリは、note有料記事の購入者特典として配布できる静的Webアプリです。

ZIPファイルを解凍し、`index.html` をブラウザで開くだけで使えます。

外部APIやサーバーは不要です。
入力内容や保存リストは、利用者のブラウザ内の `localStorage` に保存されます。
外部サーバーには送信されません。

## 購入者向けの基本操作

1. ZIPファイルを解凍します。
2. `index.html` をダブルクリックしてブラウザで開きます。
3. 作成モードで「1枚絵」または「4コマ漫画」を選びます。
4. テンプレート、主人公設定、舞台、絵柄、雰囲気、画像生成AIの種類などを選びます。
5. 同じ主人公を描きたい場合は、主人公タイプを選びます。
6. 4コマ漫画の場合は、1コマ目から4コマ目までの内容も選びます。
7. 「プロンプトを生成」を押します。
8. 生成されたプロンプトをコピーします。
9. 画像生成AIに貼り付けてイラストを作成します。

## スマホ・タブレットで使う方法

このアプリは、パソコンだけでなく、iPhone、iPad、Androidスマートフォン、Androidタブレットでも使用できます。

ただし、スマホやタブレットでは、ZIPファイルの解凍方法や `index.html` の開き方が端末によって少し異なります。

うまく開けない場合は、下の「うまく開けない場合」を確認してください。

### iPhone・iPadの場合

#### 事前に必要なもの

- iPhone または iPad
- Safari
- ファイルアプリ
- ダウンロードしたZIPファイル

#### 手順

1. noteの有料部分からZIPファイルをダウンロードします。
2. ダウンロードしたZIPファイルを、iPhoneまたはiPadの「ファイル」アプリで開きます。
3. ZIPファイルをタップして解凍します。
4. 解凍されたフォルダを開きます。
5. フォルダの中にある `index.html` をタップします。
6. Safariなどのブラウザで画面が開いたら、「選ぶだけイラストプロンプトメーカー」を使えます。
7. 作成モードを選びます。
   - 1枚絵
   - 4コマ漫画
8. テンプレート、主人公、舞台、絵柄、雰囲気などを選びます。
9. 「プロンプトを生成」を押します。
10. 生成されたプロンプトをコピーします。
11. ChatGPTなどの画像生成AIに貼り付けて使います。

### Androidの場合

#### 事前に必要なもの

- Androidスマートフォン または Androidタブレット
- Chromeなどのブラウザ

3. ZIPファイルを解凍します。
4. 解凍されたフォルダを開きます。
5. フォルダの中にある `index.html` をタップします。

7. 画面が開いたら、「選ぶだけイラストプロンプトメーカー」を使えます。
8. 作成モードを選びます。
   - 1枚絵
   - 4コマ漫画
9. テンプレート、主人公、舞台、絵柄、雰囲気などを選びます。
10. 「プロンプトを生成」を押します。
11. 生成されたプロンプトをコピーします。

### うまく開けない場合

`index.html` をタップしても正しく表示されない場合は、以下を確認してください。

- ZIPファイルをもう一度解凍する
- `index.html`、`style.css`、`script.js` が同じフォルダ内にあるか確認する
- フォルダ名やファイル名を変更しない
- iPhone・iPadの場合は、Safariで開き直す
- Androidの場合は、Chromeで開き直す
- iPhone・iPadの場合は、iCloud Drive内に保存してから開く
- Androidの場合は、Googleドライブ内ではなく、端末内のダウンロードフォルダなどに保存してから開く
- Androidの場合は、別のファイル管理アプリで開いてみる

### スマホ・タブレットで使うときの注意

このアプリは、画像生成AIそのものではありません。

画像を直接作るアプリではなく、画像生成AIに貼り付けるためのプロンプトを作るアプリです。

生成されたプロンプトは、コピーしてChatGPTなどの画像生成AIに貼り付けて使用してください。

保存リスト機能は、ブラウザ内の保存機能を使っています。

スマホやタブレットでは、ブラウザやファイルの開き方によって、保存内容が消える場合があります。

大事なプロンプトは、コピーしてメモアプリなどにも保存しておくと安心です。

## 作成モード

- 1枚絵: 1枚のイラスト生成用プロンプトを作成します。テンプレートの `singlePrompt` をもとに、場面説明を自動生成します。
- 4コマ漫画: 1コマ目から4コマ目までの内容を選び、4コマ漫画用プロンプトを作成します。

1枚絵モードでは、4コマごとの内容選択と最後の印象欄は非表示になります。

4コマ漫画モードでは、各コマの内容選択と最後の印象欄が表示されます。

## 使い方

1. `index.html` をブラウザで開きます。
2. 作成モードを選びます。
3. テンプレートを選びます。
4. テーマ、登場人物、舞台、絵柄、雰囲気、用途、画像生成AIの種類などを選びます。
5. 必要に応じて主人公設定や追加メモを編集します。
6. 同じ主人公を描きたい場合は、主人公タイプを選びます。
7. Instagram向けに使う場合は、「Instagram用サイズ」ボタンを押すと、用途とサイズ・比率が自動で設定されます。
8. 「プロンプトを生成」を押します。
9. 完成プロンプトを確認し、「コピー」で画像生成AIに貼り付けます。

## 各項目の説明

- テンプレート選択: 日常漫画、4コマ漫画、やさしいイラスト、ビジネス挿絵、風景写真風、ファンタジー、シンプル背景、SNS投稿向け、ブログ挿絵、自由テーマから選べます。テンプレートは「どんな用途・構図で描くか」を決める項目です。選んだテンプレートに合わせて、テーマ候補、舞台、雰囲気、用途、最後の印象などの初期設定を自動反映します。
- 作成モード: 1枚絵と4コマ漫画を切り替えられます。
- テーマ: 朝の出来事、仕事中、休憩時間、買い物中、家族との時間など、日常のシチュエーションを選ぶ項目です。テンプレートごとに候補が切り替わります。自由入力もできます。
- 伝えたいこと: 笑い、涙、癒し、疲れ、学び、共感、驚き、勇気、懐かしさ、前向きな気持ちなどを選べます。テーマを選ぶと自然に調整されます。自由入力もできます。
- 画像生成AIの種類: ChatGPT、Gemini、Imagen、Midjourney、Flux、その他（汎用）から選べます。自由入力もできます。
- 主人公設定: 年代、性別・雰囲気、性格、表情、服装、髪型、役割をリストから選べます。各項目は自由入力にも対応しています。
- 主人公設定文の自動生成: 選択内容をもとに、画像生成AI向けの主人公設定文を自動で組み立てます。
- 主人公タイプ: オリジナル、人物ルール、基準画像、人物ルール＋基準画像から選べます。
- 人物ルール参照: キャラクターの顔・服装・雰囲気を統一したい場合に、人物ルールを参照するための共通指示文をプロンプトへ追加できます。チェックを入れた場合は、Chatに人物ルール画像を貼り付けてから生成プロンプトを使ってください。
- 基準画像参照: 同じ主人公を描きたい場合に、基準画像（マスターキャラクター）を最優先で参照する指示文をプロンプトへ追加できます。
- 他の登場人物: 小さな案内役、家族、同僚、医師、店員などを選べます。
- 舞台: 自宅の部屋、カフェ、職場、学校、公園、森、海、山、商店街などを選べます。
- 4コマごとの内容選択: 4コマ漫画モードで、1コマ目から4コマ目までの内容パターンを個別に選べます。
- 各コマの自由入力: 各コマで「自由入力」を選ぶと、そのコマの内容を直接入力できます。
- 最後の印象: 4コマ漫画モードで、最後の雰囲気を決めます。少し笑顔になる、ほっとする、気づく、安心する、一件落着などを選べます。自由入力もできます。
- 絵柄: やさしい手描き風、エッセイ漫画風、セルアニメ風、水彩画風、色鉛筆風、写真風などを選べます。絵柄は「どんな画風で描くか」を決める項目で、テンプレートとは独立して変更できます。作品名、作家名、会社名、ブランド名を使わない表現にしています。自由入力もできます。
- 色・雰囲気: 明るい、静か、切ない、幻想的、季節感があるなどを選べます。自由入力もできます。
- 用途: note、Instagram、X、ブログ、YouTubeサムネイル、プレゼン資料などを選べます。自由入力もできます。
- サイズ・比率: 16:9、noteヘッダー、正方形、縦長などを選べます。自由入力もできます。
- Instagram用サイズ: Instagram投稿、リール、ストーリーズ、横長投稿向けのサイズをボタンで指定できます。
- 必須の小物（必要な場合のみ）: 基本的な小物はAIが舞台に合わせて自然に選択します。この項目では、ストーリー上どうしても必要な小物だけ、チェックまたは自由入力で追加してください。舞台を変更すると、舞台に合う小物候補へ自動で切り替わります。不要な小物を増やすほど、画像生成AIが意図しない補完を行いやすくなります。
- 避けたいこと: 文字化け、英語表記、主人公が若すぎる、リアル写真風などを複数選択できます。自由入力で独自の避けたいことも追加できます。舞台を変更すると、舞台に合わない小物や表現を避ける項目も自動で追加されます。選んだ内容と矛盾する避けたいことは、プロンプト生成時に自動で調整されます。
- 追加メモ: 構図やセリフ量、読者向けの細かい希望を書けます。

## テンプレート別テーマ候補

テンプレートを選ぶと、そのテンプレートに合うテーマ候補へ自動で切り替わります。

たとえば「風景写真風」を選ぶと、晴れの日、雨の日、夕暮れ、朝、夜、春、夏、秋、冬など、風景向けのテーマ候補が表示されます。

テーマを選ぶと、伝えたいこと、舞台、色・雰囲気、必須の小物、避けたいことも自動で調整されます。

風景写真風などのテンプレートでは、小物を少なめにして、舞台に自然に合う構図になるようにしています。

## 舞台別の小物・避けたいこと

舞台を選ぶと、「必須の小物（必要な場合のみ）」の候補と「避けたいこと」の候補が自動で切り替わります。

たとえば、カフェを選んだ場合は、コーヒーカップやメニューなどの候補が表示されます。

同時に、会社のオフィス風にしない、学校の教室風にしない、ノートPCや作業機器を勝手に追加しない、などの避けたいことが追加されます。

共通の避けたいことは常に表示されます。

舞台を変更しても、ユーザーが手動でチェックした避けたいことは、同じ候補が残っている範囲で保持されます。

ノートPC、デスクトップPC、大型モニター、キーボード、タブレット、スマートフォンなどのデジタル機器は、初期候補には入れていません。

必要な場合だけ、「その他の小物」に自由入力で追加してください。

たとえば、4コマの内容で「スマホで実際に試す」を選んだ場合は、スマートフォンを描かないという意味に読める避けたいことは自動で外れます。

ただし、ノートPC、デスクトップPC、大型モニターなど、今回の指示にない機器は引き続き描かないように出力されます。

## キャラクターを安定させる使い方

同じ主人公を繰り返し描きたい場合は、基準画像と人物ルールを使うと安定しやすくなります。

### 基準画像とは

基準画像は、主人公の見た目のもとになる画像です。

顔、髪型、輪郭、体型、服装、色、全体の雰囲気をそろえたい場合に使います。

画像生成AIには、この基準画像を最優先で見てもらうように指示します。

### 人物ルールとは

人物ルールは、主人公を描くための補足資料です。

三面図、表情サンプル、カラーパレット、NG例などをまとめた資料を想定しています。

基準画像だけでは伝わりにくい表情、色、避けたい描き方を補うために使います。

### おすすめの貼り付け順

キャラクターを安定させたい場合は、Chatに以下の順番で貼り付けてください。

1. 基準画像（マスターキャラクター）
2. 人物ルール
3. このアプリで生成したプロンプト

この順番にすると、画像生成AIが「何を最優先にすればよいか」を判断しやすくなります。

### キャラクターを安定させるコツ

- 主人公タイプは「人物ルール＋基準画像」を選ぶ
- 基準画像は、顔や服装が分かりやすいものを使う
- 人物ルールには、OK例だけでなくNG例も入れる
- プロンプトでは、変更してよい部分と変更してはいけない部分を分ける
- 表情、ポーズ、背景、小物は変えてよいと伝える
- 顔、髪型、服装、色、雰囲気は変えないと伝える
- 生成結果がぶれた場合は、基準画像と人物ルールをもう一度貼り直す

## 自由入力とコピー

主要な選択項目には「自由入力」を用意しています。

リストにない内容を使いたい場合は「自由入力」を選び、表示された入力欄に内容を書いてください。

生成したプロンプトをコピーすると、成功時にコピーボタンが一時的に「コピーしました！」へ変わります。

コピーできない場合は、完成プロンプト欄を手動で選択してコピーしてください。

## 保存リスト機能

「保存名」に名前を入力してから「入力内容を保存」を押すと、現在の選択内容と生成済みプロンプトをブラウザの `localStorage` に保存します。

保存名が空の場合は、現在選択中のテーマ名を保存名として使います。

保存データは複数件保存できます。1件だけを上書きするのではなく、保存するたびに新しい保存データとして追加されます。

「保存したプロンプト一覧」では、保存名、モード、テンプレート名、テーマ、用途、保存日時を確認できます。

一覧の「読み込み」を押すと、その保存データの入力内容と生成済みプロンプトを復元できます。

一覧の「削除」を押すと、不要な保存データを削除できます。削除前には確認画面が表示されます。

保存内容は同じブラウザ内にだけ残ります。外部サーバーには送信されません。

## 公開用と個人用

公開用は、誰でも使いやすい汎用テンプレート中心の構成です。

個人用にしたい場合は、`script.js` の `templates`、`options.theme`、`defaultHeroSettings` を自分用に差し替えてください。

主人公の初期値は、`script.js` の `defaultHeroSettings` で管理しています。個人用にする場合は、ここを自分のブランドガイドに合わせて変更できます。

公開用・配布用には、個人的な具体名や制作履歴を入れすぎない方が使いやすくなります。

個人用にカスタマイズする場合は、公開用とは別のフォルダーや別リポジトリで管理すると安全です。

## 必要に応じてGitHub Pagesで公開する方法

1. 公開用の4ファイルと `LICENSE` をGitHubリポジトリに置きます。
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
   - `LICENSE`
2. GitHubのリポジトリ画面で `Settings` を開きます。
3. `Pages` を開きます。
4. `Build and deployment` の `Source` で `Deploy from a branch` を選びます。
5. 公開したいブランチを選び、フォルダは `/root` を選びます。
6. `Save` を押します。
7. 数分後に表示されるURLからアプリを開けます。
