# 4コマプロンプトメーカー

## このリポジトリについて

このフォルダーは個人用リポジトリ `yonkoma-prompt-maker-personal` です。

公開用の汎用版は `yonkoma-prompt-maker` に分けています。

## アプリ概要

4コマ漫画の画像生成AI向けプロンプトを、リスト選択だけで作成できる静的Webアプリです。

note記事の挿絵やヘッダー、SNS投稿用の4コマ漫画を作るときに、毎回プロンプトを手でコピーして変更部分を書き換える手間を減らすことを目的にしています。

外部APIは使いません。HTML、CSS、JavaScriptだけでブラウザ上で動作します。

## 使い方

1. `index.html` をブラウザで開きます。
2. テンプレートを選びます。
3. テーマ、登場人物、舞台、展開パターン、オチ、絵柄などを選びます。
4. 必要に応じて主人公設定や追加メモを編集します。
5. 「プロンプトを生成」を押します。
6. 完成プロンプトを確認し、「コピー」で画像生成AIに貼り付けます。

## 各項目の説明

- テンプレート選択: よく使う4コマテーマの初期設定を自動反映します。
- テーマ: 4コマ漫画の中心になる話題です。自由入力もできます。
- 伝えたいこと: 笑い、共感、学びなど、読後感を決めます。
- 主人公設定: ブランドガイド準拠の50代男性設定です。必要に応じて編集できます。
- 他の登場人物: AIアシスタント、家族、同僚、医師などを選べます。
- 舞台: 自宅、パソコン前、スーパー、病院などを選べます。
- 4コマの展開パターン: 物語の流れを決めます。
- 4コマごとの内容選択: 1コマ目から4コマ目まで、それぞれの内容パターンを個別に選べます。
- 各コマの自由入力: 各コマで「自由入力」を選ぶと、そのコマの内容を直接入力できます。
- オチ: 最後に伝える気づきや学びを決めます。
- 絵柄: 手描き風、エッセイ漫画風などを選べます。
- 色・雰囲気: note向けの落ち着いた色やブランドカラーを選べます。
- 用途: note、X、Instagram、ブログなどの使用先を指定します。
- サイズ・比率: 16:9、noteヘッダー、正方形、縦長などを選べます。
- 入れたい要素: ノートPC、スマホ、吹き出し、4コマ枠などを複数選択できます。
- 避けたいこと: 文字化け、英語表記、主人公が若すぎる、メガネ、ひげなどを複数選択できます。
- 追加メモ: セリフ量や読者向けの細かい希望を書けます。

テンプレートを選ぶと、テーマやオチだけでなく、1コマ目から4コマ目までの初期選択も自動で反映されます。

## 保存機能

「保存名」に名前を入力してから「入力内容を保存」を押すと、現在の選択内容と生成済みプロンプトをブラウザの `localStorage` に保存します。

保存名が空の場合は、現在選択中のテーマ名を保存名として使います。

保存データは複数件保存できます。1件だけを上書きするのではなく、保存するたびに新しい保存データとして追加されます。

「保存したプロンプト一覧」では、保存名、テンプレート名、テーマ、用途、保存日時を確認できます。

一覧の「読み込み」を押すと、その保存データの入力内容と生成済みプロンプトを復元できます。

一覧の「削除」を押すと、不要な保存データを削除できます。削除前には確認画面が表示されます。

保存内容は同じブラウザ内にだけ残ります。外部サーバーには送信されません。

## GitHub Pagesで公開する方法

1. この4ファイルをGitHubリポジトリに置きます。
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
2. GitHubのリポジトリ画面で `Settings` を開きます。
3. `Pages` を開きます。
4. `Build and deployment` の `Source` で `Deploy from a branch` を選びます。
5. 公開したいブランチを選び、フォルダは `/root` を選びます。
6. `Save` を押します。
7. 数分後に表示されるURLからアプリを開けます。
